 Exporting an instrument to a file:

Exporting an instrument:

The service allows you to export your instruments to files in your local system. The files will contain the information
about the instrument in XML format.
The main utility of exporting an instrument is sharing it, this is because an exported instrument can be imported using
the file generated.

To export an instrument:

1. We have to save the changes in our instrument, this is because the file will only contain the last saved changes
(First we press the save button)

(We have to save everytime we want a change to be updated in the database)

 2. Once the instrument is saved, we have to press the export button, when pressed a system explorer window will be open and will allow to assign a name to our file

Press the export button :


A warning window will be displayed, if we want to continue we press Accept



A window of our system explorer will open



3. A new file will be created in our system, stored in the localization selected. The file's extension is always .evx

The exported file could be used to share our instrument with other people, using the import option, or in order to have a security copy.
