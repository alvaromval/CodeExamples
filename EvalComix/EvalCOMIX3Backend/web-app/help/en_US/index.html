 Introduction:

EvalCOMIX 3 is a service developed by EVALFor Research group at Cadiz's University (UCA) and several people have participated.

Description:
The assessment service is based on REST, XML and Shockwave/flash and offers the possibility of creating assessment
instruments and making assessments using some instrument patterns like control lists, rubrics, etc.
Due to the service use shockwave/flash and xml, the instruments and assessments can be obtained in both formats (although the shockwave/flash format offers a visual interactive interface)
The service can be integrated in any tool that follows the service API, adding the assessment functionality in an external and transparent way. The client will manage its users and the instruments and assessments created with the service.
Basically, the service offers a visual interface that allows to create assessment instruments, making assessments over these instruments and saving the instruments and assessments instances using identifiers provided by the user. Doing this, the service's client only has to worry about managing these identifiers, like a new table in his database.

EvalCOMIX 3 service is easy to integrate in e-learning environments and add a complete assessment functionality. At present EvalCOMIX 3 has been integrated in Moodle and is being integrated in LAMS.
