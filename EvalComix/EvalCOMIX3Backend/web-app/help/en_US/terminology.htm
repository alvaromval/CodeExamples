 Terminology:

EvalCOMIX 3 service offers some types of instruments to create adaptable assessments to any king of activity that
can be assessed, this instruments are:
  • Control list
  • List/set of values
  • Control list + value list
  • Rubric
  • Decision matrix
  • Mix tool
These instruments share common elements:
  • Dimensions: The different aspects or facets that compound the assessment object.
  • Subdimensions: The different aspects or facets that compound a dimension.
  • Attributes Each one of the elements that globally compound a subdimension.
  • Set of values: scale used for assessing/weighing up each one of the dimensions/subdimenions and/or attributes, and the management level of the students.
  • Assessments: each one of the grades/scores estimated over the attributes.
  • Percentages: Each one of the percentages assigned to any of the previous elements.
  • Grade: Transformation of all the previous assessments (with their percentage) to a unique score, normally between 0 and 100.
