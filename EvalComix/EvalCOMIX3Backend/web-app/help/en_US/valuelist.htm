 List/set of values:
A Set of values or Value list is composed by attributes and a group of categories that assess the grade of fulfillment of a
characteristic, performace or feature of the item represented.
While a control list gets to determine the presence or absence of an attribute, using set of values we can identify the frequency
of an attribute or performance.
