 Assessment:

All the instruments in the service can be used for assessment except Decision Matrix, because is an instrument to make decisions.

The asseesment is done using the evaluation screen of the instruments created, that looks similar to the edition screen but
without the editing options.

i.e. If we use a List/Set of values, the edition will look like this:

And the evaluation will look like this:


When we are in the evaluation, we only can modify the attributes of subdimensions or global assessments. Setting the appropraite values:


As we modify the values, we can see how the grade change. The grade is on the top right corner of the screen:


In all instruments, values are between 0 and 100, divided into the number of values, i.e., in the example showed above, be have 6 values
in the subdimension, so "Value0" is 0, "Value1" is 16% ... and "Value5" is 100%.
Besides we have to consider the percentage of the element that owns the values, in this case, each attribute has a 16% of value in the subdimension.

Percentages are always relative to the elements in the same level, and their total is always 100%.
Let's see in the example above:
The dimension and the global assessment -> each one has a 50% (of the parent element, in this case the List of values)
The subdimension -> because there is onlye one, has a 100% (of the parent element, in this case the dimension)
The attributes in the subdimension -> each one has a 16% of the subidmension)
(Percentages can be always modified by the user)


In this example:
1. If all the attributes are set to the greater value ("Value5"), the subdimension will have a value of 100%
2. Because there is only one subdimension, in this case the subdimension is the 100% of its parent element (the dimension)
3. If the global assessment is set to the minimum value "Value0", the value of the global assessment will be 0%
4. The final grade will be 50, because the value of the dimension is 100 and the value of the global assessment is 0 (and each element has a 50%)


WARNING: After the assessment we have to save to update the results in the database.

Assessment with rubrics

In this case, the assessment with rubrics is a little different because values have the final value already set. In this case we have no upward lists of
values, we have cells with a values.


If we select 20, the value of the attribute will be 20 (afterwards we have to consider the percentage...), if we select a cell with 100, the value will be
100, etc. So rubrics are no limited to upwards lists of values, like the other instruments, because we can specify any range of values (always between 0 and 100).
