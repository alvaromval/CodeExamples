Importing an instrument:

Import an instrument consists of recovering it from a file, previously created using the export option.

The importation can be done in two ways:

- Before creating the assessment instrument
- When we are creating the assessment instrument

1. Before creating the instrument
When we are going to create a new instrument, an you are not already selected an instrument type, a selection window will be displayed. We choose "Import an instrument"


If we press Accept, a system explorer window will open and allow us to select a file with extension .evx (this file contains information about an assessment instrument)



If we continue, we will see a screen with the instrument imported
WARNING: If we want to store the imported instrument, we have to save after the importation

2. We can also import an instrument over an instrument we are editing, to do this we have to press the importation button :


A system explorer window will open, and allow us to select a .evx filetype


If we continue, the screen will update with the new instrument
WARNING: If we want to update the changes, we have to save after the importation
