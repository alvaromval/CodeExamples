 Control list + Value list:
A Control list + Value list is a kind of Control list in where if an attribute is fulfilled, it can be assessed using a set of values.

Creating a control list+value list

To create a control list+value list we have to follow the next steps:

An empty control list+value list is like this:

We have to introduce a title, a number of dimensions and we have to choose if the instrument is going to have a global assessment and its number of values
(the global has by default the same value as a dimension)

When finished, click . This button has to be clicked every time we want to insert new elements or update changes in the fields:

In this case we have introduced one dimension and it's going to have a global assessment with 5 values:

As we can see, the dimension has been added empty, although the global assessment is already complete, displaying a list with 5 values (from 0 to 100, Value0 is 0
and Value4 is 100).
The dimension has 4 fields:
- Dimension: to assign a name
- Nº subdimensions: the number of subdimenions in the dimension
- Nº values: Numbre of values in the list (two for the control list, Yes and No, and any for the value list)
- Global assessment: is like the global assessment before, but in the dimension.

We are going to add 1 subdimension, and a value list with 5 values (the control list values are created by default). Also we want a global assessment in the dimension:
(click to update changes):

The screen shows the dimension with two sets, one for the control list with two values (Yes and No) and other for the value list, with 5 values (from 0 to 100)
We also have a global assessment in the dimension (in green) with the same numbre of values than the value list (this can not be modified).

The subdimension is the last nested element.  In subdimensions, we have to add attributes in order to complete the instrument (the attributes are used to make the
assessment). We set a number of attributes in "Nº attributes" and click

The attributes are divided by two colums, one for the control list and another for the value list. If attrubutres in the control list are set to "No", values in the value
list will remain disabled.

One we have the instrument, we have to assign appropriate names to the elements and save the changes.

WARNING: In order to update changes in the database, we always have to save changes before closing the instrument.
