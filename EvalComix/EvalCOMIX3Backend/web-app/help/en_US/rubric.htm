 Rubric:
Rubric or evaluation matrix, is an instrument, just like the value list, that enables to assess the level of fulfillment of a characteristic,
feature, performance or attribute represented as an item.
The main difference between the rubric and the value list is the requirements' description and the capacity of specify different scales
inside each assessment.
Rubric are assessment guides normally used in the evaluation of students' performace, in order to specify what is expected in the students' work,
value their work and make easier the feedback.
