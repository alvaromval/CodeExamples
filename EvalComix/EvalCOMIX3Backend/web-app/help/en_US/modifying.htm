 Modifying an instrument:

When we have already an instrument created, o when we are editing one, we can undo easily the changes
made.

i.e. In a simple control list, like the showed below:


Once we have already stablished the number of subdimensions, and they have been already added to the instrument, we want
to modify again them, i.e. now we want two subdimensions.
We can do this in two different ways:
1. Pressing , in the bottom left corner, in the dimension.
2. Modifying the number of subdimensions to 2 and pressing in the dimension

In the next image we can see the two options marked in green:


As we use one option or another, is the same:


A new empty subdimension is added just after the old one. The new element added will always have an average percentage, considering the number
of elements of its type in the parent element (in this case the dimension).
In this case, because we have two subdimensions (after adding the new one), each one will have a percentage of 50%. (if we add two more
instead of one, they will be added with a 33%)

This process is valid to all instruments in the service and to all elements in the instruments: subdimensions, dimensions, attributes.
