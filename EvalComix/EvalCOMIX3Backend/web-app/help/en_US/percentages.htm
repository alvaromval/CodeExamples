 Setting elements percentages:

Modifying a percentage
Elements percentages are describe in the percentage cells  (in compound elements, like dimensions) and
(in simple elements, like attributes)

The percentage indicate the value of the element inside its parent element, i.e., we have one subdimension with 5 attributes in a List/Set of values:

Each attribute's percentage is set to 20% (because we have 5). But we can modify the percentage of any of them,  for example, if we consider
one attribute more important than the others.
To do this, we set the new percentage in the cell:

After setting the new value (between 0 and 100) we press . Immediately, the other percentages are adapted to the new value:

We can see the rest of percentages set to 16%.

Modifying various percentages at once

If we want to modify various percentages at the same time (always between elements of the same type: between attributes, between subdimensions...)
I.e. we have three subdimensions (by default 33% each one) and we want to set the percentage of the first subdimension to 50% and the second to
40%.
If we do this one after one, first we set the percentage of the first sudbimension to 50%, and the result is:


The percentage of the first subdimension has been set to 50%, and in the other two, has been adapted (25% each one)

If now we set the percentage of the second subdimension to 40%, we will not have the result we are expecting:


Percentages has been set to 40-40-20 ¿Why is this? Because the first subdimension had a 50% and the third a 25%, so if we maintain the proportion (2/1),
we will have a 40% in the first subdimension and a 20% in the third one.
So, ¿how can we do this easily?
The solution is set the desired percentages at the same time, the 50% in the first subdimension and the 40% in the second.
We will do this in this way:

1. We change the percentage of the first subdimension to 50% but we do not press to update
2. We change the percentage of the second subdimension to 40% and now we can press in any of the modified subdimensions.

Automatically, the percentages are set to the desired values:


WARNING: In order to update changes in the database, we always have to save changes before closing the instrument.
