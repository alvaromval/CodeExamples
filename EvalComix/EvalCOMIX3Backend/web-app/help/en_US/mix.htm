 Mix tool:
A mix tool is an instrument composed by n assessment instruments and its grade is the average of the instruments grades that
compose it.

Creating a mix tool

A mix tool can be created in two ways, choosing a instruments already created, selecting empty instruments:

In example, in the image below we can see a mix tool composed by two instruments already created, a value list and a rubric;

This is equivalent to joining instruments, one after one. This way we can create instrument as complex as we want.

Other way to create a mix tool is selecting a list of instrument types, in order to create an empty mix tool:
