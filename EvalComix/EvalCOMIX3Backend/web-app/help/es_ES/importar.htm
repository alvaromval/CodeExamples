 Importar un instrumento:

Pasos a seguir para importar un instrumento:

Importar un instrumento consiste en recuperar un instrumento desde un fichero que alguien ha creado previamente
utilizando la opción de exportación.

La importación se puede realizar en dos momentos distintos:

- Antes de crear el instrumento
- Cuando estemos creando un instrumento

1. A la hora de crear el instrumento
Se nos abrirá una ventana con las opciones de creación de instrumentos. Seleccionamos "Impotar instrumento desde fichero"

Si pulsamos aceptar se nos abrirá una ventana del explorador que nos permitirá seleccionar un fichero de tipo .evx (que contiene la información del
instrumento), para importarlo.

Si continuamos, se nos mostrará una pantalla con el instrumento importado.
IMPORTANTE: Si queremos almacenar el instrumento importado, debemos presionar el botón de guardado.

2. Tambiém podemos importar un instrumento sobre un instrumento que ya estemos creando, presionando para ello el botón de importación

Se nos abrirá una ventana del explorador que nos permitirá seleccionar un fichero de tipo .evx

Si presionamos aceptar, la pantalla se actualizará con la información del fichero importado.
IMPORTANTE: Siempre que importemos debemos presionar el botón de guardado, con el fin de que se almacenen los cambios realizados.
